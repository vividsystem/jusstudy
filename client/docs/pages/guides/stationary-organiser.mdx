## Creating The Base

Now I'll introduce you to the sketch tool on the top-left. Click it and you will see the following options appear:

![Sketch toolbar](https://github.com/user-attachments/assets/b39a6ba4-5607-4765-b188-747d4f20405f)

In CAD software, you usually create a 2D sketch first and then extrude it into a 3D object.

Let's start by selecting the rectangle tool and creating a square:

![Creating square sketch](https://github.com/user-attachments/assets/1be666f8-83e0-4e2a-ad60-c4ec39bbfeb7)

## Extruding The Shape

Now let's make it 3D!

Select the **Extrude** tool at the top left. It will ask you to select a plane, then select the area inside the square.

Remember to right-click and exit rectangle mode first, otherwise you won't be able to select the shape.

You can adjust the height however you want.

(Just don't make a pen holder the height of a skyscraper 😭)

![Extruding the square](https://github.com/user-attachments/assets/c39e4e4f-7c5b-4905-87fa-b3ada2e7ff72)

## Creating The Holder

Repeat the same process and create another sketch on top of the structure:

![Top sketch](https://github.com/user-attachments/assets/8ce41880-1824-43b4-8748-0668f09c56de)

Instead of extruding upwards, select the **Remove** option from the left panel (which is set to **Add** by default).

This will remove material instead of adding it.

Adjust the height and now we have a simple box with a hole!!!

Congrats 🎉

![Removing material](https://github.com/user-attachments/assets/7a5d05c4-8769-4228-af7d-eda4a004bea6)

![Result](https://github.com/user-attachments/assets/3d6885d0-fbe0-4399-83bb-555b9e0c9a35)

![Holder shape](https://github.com/user-attachments/assets/9b0bee2f-d386-4d38-afbc-0a89d789628c)

## Adding Rounded Edges

Select the **Chamfer** tool from the top bar and choose the edges you want to modify.

Chamfer adds a slanted edge instead of a sharp edge.

The **Fillet** tool does something similar but creates rounded edges.

I recommend playing around with both to see how they improve your designs!

![Chamfer tool](https://github.com/user-attachments/assets/d252e009-7c5c-4106-9c58-422a550caabc)

## Measuring Dimensions

Another useful tool is the **Measure** tool on the top-right.

It allows you to measure and adjust the size of sketches.

![Measure tool](https://github.com/user-attachments/assets/05fc1315-84d5-40ba-8e0c-6056f903902b)

![Measuring](https://github.com/user-attachments/assets/6e75f9c0-2869-40bf-8afa-9122da3d3ce2)

## Creating The Overhang

Now let's create an overhang on the right side.

Use the sketch tool again, select the correct plane, draw your shape, and extrude it.

This is a great way to get familiar with CAD workflows.

![Overhang](https://github.com/user-attachments/assets/4730d394-2f7b-4fdd-9876-37dc164b71fe)

## Adding A Circular Hole

Now let's create a circular hole.

Select a plane and use the **Circle** tool instead of the rectangle tool.

Create a circular sketch and extrude it using the **Remove** option.

This will cut away material and create an empty space.

Use the Fillet tool again to smooth the edges.

And here we have our completed stationary organizer!

![Final model](https://github.com/user-attachments/assets/fb2a77cf-5386-42c1-86da-8e96ef686fbd)

![Model view](https://github.com/user-attachments/assets/d86a499f-3b70-496c-9c0e-79f349f0d062)
